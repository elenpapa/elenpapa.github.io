This directory should contain images for the painted books page.

### Hero Section

- `hero-painted-book.png` - Main hero image showing painted book edges with detail

### Gallery Items (Book Examples)

- `book-1.png` - Το Όνομα του Ρόδου by Umberto Eco
- `book-2.png` - Περηφάνια και Προκατάληψη by Jane Austen

- `book-5.png` - Εκατό Χρόνια Μοναξιά by Gabriel García Márquez

## Image Guidelines:

- Use high-quality photos of actual painted book edges
- Recommended dimensions: 800x600px for gallery items, 1200x800px for hero
- Format: PNG or WebP preferred
- Show clear details of the painted artwork
- Include variety of styles and themes
